# 💬 VoxCore Wiki

Welcome to **VoxCore**, a modern and fully customizable chat system for Minecraft servers.

---

## ✨ Features

- Multi-channel chat system
- Private messaging with reply support
- Advanced chat formatting
- Anti-spam & filtering system
- Chat logs
- Fully customizable
- Lightweight & high performance

---

## 🚀 Quick Start

1. Download VoxCore
2. Place it in your `/plugins` folder
3. Restart your server
4. Configure `config.yml`
