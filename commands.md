# 🧾 Commands & Permissions

## Commands

| Command | Description |
|--------|------------|
| /msg | Send private message |
| /reply | Reply to last message |
| /ignore | Ignore player |
| /chat reload | Reload config |

---

## Permissions

| Permission | Description |
|------------|------------|
| voxcore.admin | Full access |
| voxcore.chat | Use chat |
| voxcore.staff | Staff chat |
